Sealed Air Projects:


Here are some of my most noteworthy projects I worked on at Sealed Air. Click a project to find descriptions and links for that specific project.
  • Product Selector
    Allows a user to filter through over 250 products until they narrow their selection to the product or group of products that they are looking for (results can be seen after the first 2 filtration selections). Additionally it contains a sidebar dropdown menu to filter further. Images can be selected and will display more valuable information on the products.
    See it in action here: http://www.sealedairdigital.com/natools/productselector/
  • Dilution Calculator
    Allows a user to filter through dilution control platform (previously called brand), product category, products, and the product's dilution ratio before entering a price per case to apply the the previous selections. The application then uses the user selected dilution ratio and the price per case combined with the bottles per case (associated with the product), computes end use quarts and end use gallons, and displays the information in a chart below the selections.
    See it in action here: http://www.sealedairdigital.com/natools/dilutioncalculator/
  • Floor Care Solutions
    Contains three sections: Troubleshooting (index page automatically transfers the user to this page), Substrates, and Pad Selection.
    - Troubleshooting displays numerous pictures of issues that Diversey products are designed to help with. If you click on an image it will list a discription of the problem, a solution, and often shows the product best suited to help with the issue.
    - Substrates is set up similarly to Troubleshooting except when an image is clicked on, the new page displays a description, key notes, and maintenance techniques. Some of these pages give the option to scroll through images as these floor types often look very different.
    - Pad Selection uses an iframe to display a 2 page pdf containing pad information and recommendations.
    See it in action here: http://www.sealedairdigital.com/natools/FloorCareSolutions/
