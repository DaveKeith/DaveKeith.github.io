Ruby on Rails Projects:

  • Inside the Park
    Allows a user to create an itinerary and fill it with events (baseball games) extracted from an API. The user could choose a day and an event from that day, which would allow the application to show the next day’s events, continuing until the user stops adding events. The user will then be able to view their itinerary with all the details related to the events scheduled for each day.
    See it on github here: https://github.com/insidethepark/RB

    Step 6: After you have all your games, an itinerary is displayed
  • Müdr
    Allows a user in an unhappy mood to make a post, and have others make cheerful comments on their post. Comments could then be ranked and top commenters would be placed on a leaderboard.
    See it on github here: https://github.com/mudr/backend
  • Blog App
    A blog site that allows a user to create posts, allows for users to comment and/or vote on both posts and other comments. Also contains a page to see the user profile with stats on activity and popularity in relation to posts, comments, and votes. Utilizes the materialize gem for front-end styling. It was a solo project in which I designed both the front-end and back-end.
    See it on github here: https://github.com/DaveKeith/blog_app
